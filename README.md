# mahendra

Mahendra Constructions - Professional construction company website built with HTML, CSS, and JavaScript.

## Features
- Modern, responsive design
- Service showcase
- Project portfolio
- Client testimonials
- Contact forms
- Premium aesthetics with professional branding

## Project Structure
```
mahendra/
├── src/
│   └── images/          # Image assets
├── index.html           # Main HTML file
├── style.css           # Stylesheet
├── script.js           # JavaScript functionality
└── README.md           # Project documentation
```

## Technologies Used
- HTML5
- CSS3 (with Google Fonts: Vidaloka & Poppins)
- Vanilla JavaScript
- SVG logos
